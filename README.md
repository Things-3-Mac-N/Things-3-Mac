## Overview

Things 3 Professional Suite is a complete task management platform that serves productivity-focused users with the project depth, scheduling flexibility, and cross-device sync needed for a comprehensive personal productivity system. It occupies a deliberate middle ground between minimal to-do lists that lack organizational power and complex project management tools that overwhelm individual users. The result is a task manager that handles real complexity while remaining a pleasure to use every day.

Heading and checklist structure within projects enables multi-phase project management with grouped tasks under named headings and detailed subtask checklists, providing enough structure for complex projects without the overhead of a full project management application. Headings can be collapsed to focus on the current phase, and checklist items can be promoted to full tasks when they grow beyond simple checkbox items. This flexible structure adapts to projects of any complexity without imposing a rigid methodology.

Evening scheduling separates tasks into morning and evening commitment windows, enabling time-of-day planning that acknowledges that not everything on the daily list belongs in the workday and some tasks are better suited to personal time. The Upcoming view shows tasks organized by date across the coming weeks, providing a forward-looking perspective that helps distribute work evenly rather than letting it pile up. Repeating tasks handle recurring commitments with flexible scheduling rules that support daily, weekly, monthly, and custom repeat patterns.

Cloud sync via Things Cloud keeps tasks, projects, and areas consistent across iPhone, iPad, and Apple Watch with fast, reliable synchronization that makes tasks captured on any device immediately available everywhere. The sync engine is purpose-built for Things rather than relying on generic cloud storage, resulting in exceptionally fast and reliable synchronization. Apple Watch support enables task review and completion from the wrist, and Shortcuts integration allows task creation and retrieval from automated workflows.

Things 3 Professional Suite is the task management tool for users who have tried many to-do apps and settled on the one where the combination of design quality, organizational depth, and reliability makes it the system they actually maintain consistently. Its deliberate design philosophy — that a tool you love using is one you actually use — makes it the task manager that earns its place in daily routine.

## Performance & Integration

Things 3 is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Things 3 processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
